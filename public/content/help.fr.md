## Création de contenu

### Téléverser des fichiers
1. Cliquez sur le bouton **+** (FAB) dans le coin inférieur droit
2. Sélectionnez **Téléverser un fichier**
3. Choisissez vos fichiers depuis votre appareil
4. Les fichiers sont automatiquement chiffrés avant le téléversement

### Créer un formulaire
1. Cliquez sur le bouton **+**
2. Sélectionnez **Créer un formulaire**
3. Concevez votre formulaire avec différents types de champs
4. Enregistrez le modèle de formulaire pour le réutiliser

### Démarrer une conversation
1. Cliquez sur le bouton **+**
2. Sélectionnez **Nouvelle conversation**
3. Choisissez les contacts avec qui discuter
4. Envoyez des messages et des fichiers chiffrés

### Créer des dossiers
1. Cliquez sur le bouton **+**
2. Sélectionnez **Nouveau dossier**
3. Nommez votre dossier
4. Organisez vos fichiers en les faisant glisser dans les dossiers

## Contacts et invitations

### Ajouter des contacts

#### Si la personne a déjà un compte :
1. Cliquez sur **Contacts** dans la barre latérale
2. Cliquez sur le bouton **Ajouter un contact**
3. Entrez son adresse e-mail
4. Ajoutez un message personnel optionnel
5. Cliquez sur **Envoyer la demande**

**Ce qui se passe ensuite :**
- Le destinataire reçoit une notification (si activée)
- La demande apparaît dans son onglet "Demandes"
- La demande expire après 30 jours si elle n'a pas reçu de réponse
- Une fois acceptée, vous devenez tous les deux contacts et pouvez partager des fichiers

#### Si la personne n'a pas encore de compte :
1. Suivez les mêmes étapes pour ajouter un contact
2. Entrez son adresse e-mail
3. Votre client de messagerie s'ouvrira avec un e-mail d'invitation
4. Envoyez l'e-mail d'invitation pré-rempli

**Ce qui se passe ensuite :**
- Ils reçoivent un e-mail avec un lien d'inscription
- Le lien comprend votre code d'invitation
- Lorsqu'ils s'inscrivent, vous êtes automatiquement connectés
- L'invitation expire après 30 jours

### Accepter les demandes de contact
1. Cliquez sur **Contacts** dans la barre latérale
2. Allez dans l'onglet **Demandes**
3. Vous verrez les demandes en attente avec le nom et l'e-mail de l'expéditeur
4. Cliquez sur le ✓ (coche) pour accepter
5. Cliquez sur le ✗ pour refuser

### Gérer les contacts
- Consultez tous les contacts dans l'onglet **Contacts**
- Recherchez des contacts par nom ou e-mail
- Supprimez des contacts en cliquant sur l'icône de menu
- Bloquez les utilisateurs si nécessaire

### Partager avec les contacts
- Seuls les contacts peuvent voir vos fichiers partagés
- Sélectionnez les contacts lors du partage de fichiers ou de dossiers
- Les participants aux conversations sont automatiquement des contacts
- La relation de contact est mutuelle (les deux utilisateurs sont connectés)

## Partage de contenu

### Partager des fichiers
1. Faites un clic droit sur un fichier (ou cliquez sur l'icône de menu)
2. Sélectionnez **Partager**
3. Choisissez les contacts avec qui partager
4. Les destinataires reçoivent une copie chiffrée

### Partager des dossiers
1. Faites un clic droit sur un dossier
2. Sélectionnez **Partager**
3. Tous les fichiers du dossier sont partagés avec les contacts sélectionnés

### Partager des formulaires
1. Ouvrez un formulaire
2. Cliquez sur le bouton **Partager**
3. Les destinataires peuvent remplir le formulaire

### Pièces jointes dans les conversations
1. Ouvrez une conversation
2. Cliquez sur l'icône de pièce jointe
3. Sélectionnez un fichier à partager
4. Le fichier est chiffré et envoyé à tous les participants

## Gestion du contenu

### Organiser avec des dossiers
- Glissez-déposez des fichiers dans les dossiers
- Les fichiers peuvent être dans différents dossiers pour différents utilisateurs
- Votre organisation de dossiers vous est personnelle

### Favoris
- Cliquez sur l'icône étoile pour ajouter des fichiers aux favoris
- Accédez aux favoris depuis la barre latérale
- Les favoris sont privés

### Étiquettes
- Ajoutez des étiquettes aux fichiers pour une meilleure organisation
- Filtrez les fichiers par étiquettes
- Les étiquettes sont chiffrées et privées

### Renommer les fichiers
- Faites un clic droit sur un fichier
- Sélectionnez **Renommer**
- Choisissez un nom personnel (vous seul le voyez)

## Fonctionnalités de sécurité

### Chiffrement de bout en bout
- Tous les fichiers sont chiffrés avant de quitter votre appareil
- Seuls vous et les personnes avec qui vous partagez pouvez déchiffrer les fichiers
- Même nous ne pouvons pas accéder à vos données chiffrées

### Sécurité post-quantique
- Utilise ML-KEM-768 (anciennement CRYSTALS-Kyber) pour l'encapsulation de clés
- Résistant aux attaques par ordinateur quantique
- Chiffrement à l'épreuve du futur
- Standardisé par le NIST pour la cryptographie post-quantique

### Clés privées
- Votre clé privée ne quitte jamais votre appareil sans être chiffrée
- Protégez soigneusement votre phrase de passe
- Activez l'authentification biométrique pour plus de commodité

## Gestion des clés et paramètres du profil

### Méthodes d'authentification
Visitez votre page **Profil** pour gérer différentes façons de déverrouiller votre coffre-fort :

#### Protection par phrase de passe
- Votre méthode principale pour déverrouiller votre coffre-fort
- Utilisée pour chiffrer votre clé privée
- Changez votre phrase de passe à tout moment depuis le Profil
- Choisissez une phrase de passe forte et mémorable

#### Authentification biométrique
- Utilisez l'empreinte digitale ou Face ID pour un accès rapide
- Disponible sur les appareils compatibles
- Configurez depuis la page Profil
- Les données biométriques ne quittent jamais votre appareil

#### Clés de sécurité matérielles
- Utilisez YubiKey ou d'autres appareils FIDO2
- Méthode d'authentification la plus sûre
- Enregistrez les clés depuis la page Profil
- Vous pouvez enregistrer plusieurs clés en tant que sauvegarde
- Option pour stocker la clé privée directement dans le matériel (pas de phrase de passe nécessaire)
- Possibilité de basculer entre les modes d'authentification par phrase de passe uniquement et par clé matérielle

##### Modes d'authentification par clé matérielle

**Phrase de passe + Clé matérielle (Hybride)**
- Votre clé privée est chiffrée avec votre phrase de passe ET stockée dans le matériel
- Peut être déverrouillée en utilisant soit votre phrase de passe SOIT la clé matérielle
- Offre redondance et flexibilité
- Recommandé pour la plupart des utilisateurs

**Clé matérielle uniquement**
- Clé privée stockée uniquement dans les clés matérielles (pas de sauvegarde par phrase de passe)
- Sécurité maximale - pas de clé chiffrée sur le serveur
- Vous devez avoir accès à la clé matérielle pour déverrouiller
- **Important** : Enregistrez plusieurs clés matérielles en tant que sauvegarde pour éviter le verrouillage

**Basculer entre les modes**
- Supprimer la protection par phrase de passe : Après avoir stocké la clé dans le matériel, vous pouvez supprimer la sauvegarde chiffrée par phrase de passe
- Restaurer la protection par phrase de passe : Si vous utilisez le mode matériel uniquement, vous pouvez rajouter le chiffrement par phrase de passe
- Les deux opérations sont disponibles dans Profil > Section Clés de sécurité matérielles
- Vous permet de choisir l'équilibre entre sécurité et commodité

### Actions de gestion des clés

#### Générer de nouvelles clés
- Créez vos clés de chiffrement lors de la première utilisation
- Sécurisé post-quantique (ML-KEM-768)
- Gardez votre phrase de passe en sécurité - elle ne peut pas être récupérée

#### Exporter la clé privée
- Téléchargez votre clé privée en tant que sauvegarde
- Stockez-la dans un endroit sûr (hors ligne recommandé)
- Nécessaire pour récupérer l'accès si vous perdez votre phrase de passe
- Accessible depuis Profil > Section Gestion des clés

#### Importer la clé privée
- Restaurez l'accès en utilisant une clé précédemment exportée
- Téléversez votre fichier de clé depuis la page Profil
- Entrez votre phrase de passe pour déchiffrer la clé
- Utilisez ceci si vous devez récupérer votre compte

### Meilleures pratiques de sécurité
- **Ne partagez jamais votre clé privée** avec qui que ce soit
- **Gardez votre phrase de passe en sécurité** - notez-la hors ligne
- **Exportez votre clé** et stockez la sauvegarde en toute sécurité
- **Activez l'authentification biométrique** pour plus de commodité sans sacrifier la sécurité
- **Utilisez des clés matérielles** pour une sécurité maximale
- **Enregistrez plusieurs clés matérielles** pour éviter le verrouillage
- **Envisagez le mode hybride** (phrase de passe + clé matérielle) pour le meilleur équilibre entre sécurité et commodité
- **Le mode matériel uniquement** offre une sécurité maximale mais nécessite une gestion prudente des clés de sauvegarde

### Paramètres du profil
Accédez aux paramètres supplémentaires depuis votre page Profil :

- **Nom d'affichage** : Mettez à jour la façon dont vous apparaissez aux autres
- **Thème** : Basculez entre le mode clair et sombre
- **Importation JSON** : Importez des données depuis d'autres gestionnaires de mots de passe
- **Suppression du compte** : Supprimez définitivement votre compte et toutes vos données

## Trucs et astuces

- Utilisez la barre de recherche pour trouver rapidement des fichiers
- Les fichiers récents sont accessibles depuis la barre latérale
- Les fichiers partagés apparaissent dans "Partagés avec moi"
- Exportez votre clé privée en tant que sauvegarde (gardez-la en sécurité !)
- Appuyez sur Entrée dans la boîte de dialogue de dossier pour créer rapidement des dossiers
- La progression de l'importation continue même si vous naviguez ailleurs
